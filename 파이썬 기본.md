### 3. 리스트 & 리스트 컴프리헨션

#### [Python 기초] 3. 자료구조 (리스트, 튜플, 딕셔너리, 세트)

##### 1. 4대 파이썬 컬렉션(Collection) 요약

무언가를 담는 연속 자료구조 4가지의 특징 비교입니다. 코테에서는 목적에 맞는 자료구조를 선택하는 것이 핵심입니다.

| 자료구조           | 형태        | 순서  |    중복    | 변경  | 비고                                 |
| :----------------- | :---------- | :---: | :--------: | :---: | :----------------------------------- |
| **리스트(List)**   | `[1, 2, 3]` |   O   |     O      |   O   | 배열, 스택, 큐의 기본 형태           |
| **튜플(Tuple)**    | `(1, 2, 3)` |   O   |     O      | **X** | 값이 변하면 안 될 때 사용            |
| **딕셔너리(Dict)** | `{"a": 1}`  |  O\*  | 키 X, 값 O |   O   | 키(Key)로 값(Value)을 빠르게 찾을 때 |
| **세트(Set)**      | `{1, 2, 3}` | **X** |   **X**    |   O   | 교집합/차집합, 중복 제거용           |

> _참고: 파이썬 3.7 버전부터 딕셔너리는 데이터가 추가된 순서를 유지합니다._

##### 2. 리스트(List)의 활용과 시간 복잡도

리스트 하나로 배열, 스택(Stack), 큐(Queue)의 역할을 모두 수행할 수 있지만, 동작에 따라 속도(시간 복잡도) 차이가 큽니다.

```
# 1. 빠른 연산 (O(1)) - 스택처럼 쓰기
arr.append(5)  # 맨 뒤에 추가 (빠름)
arr.pop()      # 맨 끝 요소 꺼내기 (빠름)
```

```
# 2. 느린 연산 (O(N)) - 주의!
arr.insert(0, 1) # 맨 앞이나 중간에 넣기 (뒤에 있는 요소들을 다 밀어야 해서 느림)
arr.pop(0)       # 맨 앞 요소 꺼내기 (마찬가지로 다 당겨와야 해서 느림)
```

```
# 3. 정렬 메서드 비교
arr.sort()       # 원본 리스트 자체를 정렬하여 변경 (반환값 없음)
sorted(arr)      # 원본은 그대로 두고, 정렬된 '새로운 리스트'를 반환
```

##### 3. 큐(Queue)는 무조건 deque를 쓰자

리스트의 pop(0)은 너무 느리기 때문에, 코테에서 큐(Queue)나 BFS 알고리즘을 구현할 때는 반드시 collections 모듈의 deque를 사용해야 합니다.

`from collections import deque`

```
queue = deque([1, 2, 3])
queue.append(4)         # 뒤에 넣기
print(queue.popleft())  # [중요] 앞에서 빼기 (pop(0) 대신 사용, 속도 빠름!)
# 출력: 1
```

##### 4. 리스트 컴프리헨션 (List Comprehension)

반복문과 조건문을 한 줄로 줄여서 리스트를 생성하는 파이썬의 강력한 문법입니다.

```
# 일반적인 for문 사용
arr = []
for i in range(10):
    if i % 2 == 0:
        arr.append(i)

# 리스트 컴프리헨션 (위 코드를 한 줄로!)
arr = [i for i in range(10) if i % 2 == 0]
```

##### 6. 코테 치트키: defaultdict

딕셔너리에 없는 키를 조회할 때 발생하는 KeyError를 방지해 주는 아주 유용한 모듈입니다.

`from collections import defaultdict`

```
# 값이 없으면 자동으로 정수 기본값(0)으로 초기화해주는 딕셔너리 생성
d = defaultdict(int)
words = ['book', 'book', 'pen']

for w in words:
    d[w] += 1  # 원래라면 d[w]가 없어서 에러가 나지만, defaultdict라 0에서 1을 더해줌!

print(dict(d))
# 출력: {'book': 2, 'pen': 1}
```

##### 4. 코테 치트키: Counter (빈도수 계산)

문자열이나 리스트 안의 요소가 몇 번씩 등장하는지 순식간에 딕셔너리 형태로 세어줍니다.

```
	from collections import Counter

	a = 'banana'
	c = Counter(a)

	print(c)                # Counter({'a': 3, 'n': 2, 'b': 1})
	print(c['a'])           # 3
	print(c.most_common(1)) # [('a', 3)] (가장 자주 등장하는 상위 1개 요소 출력)
```

##### 5. 실전 정렬: 람다(Lambda)를 이용한 다중 조건 정렬

코딩 테스트 정렬 문제의 90% 이상은 이 lambda 식을 요구합니다. 튜플 형태로 우선순위를 지정할 수 있습니다.

```
	students = [(170, 60), (180, 80), (170, 55), (180, 75)]
	# 정렬 조건:
	# 1순위 (-x[0]): 키 내림차순 (큰 사람이 앞으로)
	# 2순위 (x[1]): 몸무게 오름차순 (키가 같다면, 가벼운 사람이 앞으로)
	students.sort(key=lambda x: (-x[0], x[1]))

	print(students)
	# 결과: [(180, 75), (180, 80), (170, 55), (170, 60)]
```

#### 5. 코테 치트키: 반복문 단짝 함수들 (`enumerate`, `zip`)

① `enumerate()` : 인덱스와 값을 동시에!

### 함수와 필수 내장 함수

#### [Python 기초] 7. 함수와 필수 내장 함수 (Functions & Built-ins)

##### 1. 함수 (def) 기본 구조

코딩 테스트에서는 반복되는 로직을 함수로 묶거나, 탐색 알고리즘(DFS/BFS)을 구현할 때 뼈대로 사용합니다.

```
	def add_numbers(a, b):
		result = a + b
		return result  # 반환값이 없으면 return은 생략 가능

	print(add_numbers(3, 5))  # 출력: 8
```

##### 2. 💡 코테 주의점: 전역 변수 (global)

> 함수 안에서 함수 밖의 변수를 **'읽어오는 것'**은 그냥 되지만, 값을 **'수정'**하려면 반드시 global 키워드로 선언해야 합니다. (백트래킹이나 DFS 문제에서 카운트를 셀 때 아주 자주 실수하는 부분입니다!)

```
	count = 0  # 전역 변수

	def dfs():
		global count  # "함수 밖의 count 변수를 여기서 수정할게!" 라고 선언
		count += 1    # 선언 없이 += 1 을 하면 에러 발생

	dfs()
	print(count)  # 출력: 1
```

##### 3. 코테 필수 내장 함수 (시간 단축 치트키)

> 직접 for문을 돌리면서 변수에 값을 저장할 필요 없이, 파이썬이 기본으로 제공하는 함수들을 쓰면 속도도 빠르고 코드도 깔끔해집니다.

```
	sum(iterable): 리스트 안의 모든 요소 합 구하기

	sum([1, 2, 3, 4, 5]) ➡️ 15

	min(iterable), max(iterable): 최솟값, 최댓값 구하기

	max([10, 20, 5]) ➡️ 20

	abs(x): 절댓값 반환 (두 좌표 사이의 거리 차이를 구할 때 필수!)

	abs(-5) ➡️ 5

	eval(expression): 문자열 형태의 수학 수식을 실제 파이썬 코드로 계산해 줌

	eval("3 + 5 * 2") ➡️ 13
```

##### 4. 수학 문제 단골: math 라이브러리

> 코딩 테스트 앞부분에서 자주 나오는 수학 문제(최대공약수 등)는 직접 식을 짜지 말고 모듈을 불러다 쓰세요.

```
import math

# 1. 최대공약수 (GCD: Greatest Common Divisor)
print(math.gcd(21, 14))  # 출력: 7

# 2. 최소공배수 (LCM: Least Common Multiple) - 파이썬 3.9 이상 지원
print(math.lcm(21, 14))  # 출력: 42

# 3. 올림과 내림
print(math.ceil(3.14))   # 출력: 4 (올림)
print(math.floor(3.14))  # 출력: 3 (내림)

```
